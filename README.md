# Rt.Lowers 公式サイト

🎸 **[Rt.Lowers Official Website](https://rtlowers.com)**  
日本のインディーズロックバンド **Rt.Lowers（ライトロワーズ）** の公式サイトです。  
ライブ情報、ニュース、ディスコグラフィー、プロフィールなどを掲載しています。

---

## 🧩 技術構成

| 分類           | 使用技術                                            |
| -------------- | --------------------------------------------------- |
| フレームワーク | [Next.js (App Router)](https://nextjs.org/docs/app) |
| CMS            | [microCMS](https://microcms.io/)                    |
| スタイル       | [Tailwind CSS](https://tailwindcss.com/)            |
| ホスティング   | [Vercel](https://vercel.com/)                       |
| 公開 URL       | [https://rtlowers.com](https://rtlowers.com)        |

バックエンドを持たず、すべてのコンテンツは microCMS から取得しています。  
ISR（Incremental Static Regeneration）により、更新内容が自動的に反映されます。

---

## 🪶 主な特徴

- microCMS で簡単にコンテンツ更新
- バックエンド不要の静的サイト構成
- Tailwind CSS によるレスポンシブデザイン
- SEO を意識したメタデータ設定
- App Router を利用した動的ルーティング
